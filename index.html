Projects

TruAlgebra



This app is designed to help students learn the basics of algebra through trial and error.

Students can either build their own equations or get a randomly generated equation to solve. The JavaScript file handles all math as well as the placement of new constants and coefficients. Students can also request hints as well as see the steps they have taken in their solving process.

This app was built with JavaScript, HTML, CSS, and Express. The formatting of the equations is handled by MathJax.

MAG Character Creator


A streamlined process for fans of Brandon Sanderson's Mistborn Adventure Game to create their character

The app guides users through the character creation process, without needing to consult the actual rulebook for options or details. Main features include:
  • When assigning stats, the app performs all necessary calculations
  • When prioritizing the character's powers, certain options for the character's race are removed, as per the rulebook
  • Upon completion, the app displays all the character info, ready to be copied down onto a physical character sheet
This app was built with JavaScript, HTML, CSS, Express, and Axios.
