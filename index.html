Projects

TruAlgebra

This app is designed to help students learn the basics of algebra through trial and error.

Algebra teachers can assign equations from a library for their students to solve. The solving mechanic includes distributing, combining like terms, hints, and an undo button.

Built using Flask for the front-end and a PostgreSQL database. The SymPy package handles the manipulation of the equations as the students solve them. Equation formatting handled by MathJax.

MAG Character Creator

A streamlined process for fans of Brandon Sanderson's Mistborn Adventure Game to create their character The app guides users through the character creation process, without needing to consult the actual rulebook for options or details. Main features include:

  • When assigning stats, the app performs all necessary calculations
  • When prioritizing the character's powers, certain options for the character's race are removed, as per the rulebook
  • Upon completion, the app displays all the character info, ready to be copied down onto a physical character sheet

This app was built with JavaScript, HTML, CSS, Express, and Axios.
